SureBetting Bot

What are sure bets ?

Sure betting is a way of placing bets with a guaranteed profit, independently from the match results, placing a precise amount of money on two opposite bets, using two (or more) different bookmakers.

Why do they guarantee a profit ?

This work because any bookmaker has its own quotes for the same bets on a match, so you can place bet A on a bookmaker and the opposite bet B on another bookmaker. If quotes on the two bookmakers satisfy some criteria, you will be guaranteed a profit. (Further explanations below)

You can find more info on different types of bets used in this program here

How this program works ? (easy explanation)

First of all, it gets all upcoming matches of the day and their quotes on a lot of bookmakers

(Up to now, it only works for bookmakers accessible from Europe, and for Under/Over bets, called ‘totals’).

The program updates the quote for each match approximately every 5 minutes.

Once it has every quote, it creates every combination for each match from different bookmakers and then it searches for a couple of opposite winning bets to place on different bookmakers (further explanations of this part below).

When it finds them, it sends a message on the telegram channel with infos of the bets to place and their profit.

APIs:

To get all the upcoming matches and their odds, this programs uses ‘the-odds-api’ APIs

You can find how these APIs works here

There are 3 types of request you can do:

  • Get every available sports: https://api.the-odds-api.com/v4/sports/?apiKey=YOUR_API_KEY
  • Get sports odds: https://api.the-odds-api.com/v4/sports/upcoming/odds/?regions=YOUR_REGIONS&markets=YOUR_MARKETS&apiKey=YOUR_API_KEY
  • Get sports scores: https://api.the-odds-api.com/v4/sports/YOUR_SPORT_KEY/scores/?daysFrom=YOUR_DAYS_FROM&apiKey=YOUR_API_KEY

In order to get all sports odds we will use the 2nd type of request, where:

YOUR_REGIONS = 'eu' (you can add more, check documentation)

YOUR_MARKETS = 'totals' (you can add more, check documentation)

YOUR_API_KEY API key given by the website

Every request will give you a .json , you can find some examples in the documentation

they represent the likelihood of an outcome occurring (in the eyes of that bookmaker). In theory, odds of 2.00 in a 2 way bet (Win or Loss being the only outcomes), represent a 50% chance of that outcome occurring.
